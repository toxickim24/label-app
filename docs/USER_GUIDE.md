## Getting Started

### Installation

**iOS:**
1. Open App Store on your iPhone/iPad
2. Search for "Label"
3. Tap "Get" to download
4. Open the app

**Android:**
1. Open Google Play Store
2. Search for "Label"
3. Tap "Install"
4. Open the app

### First-Time Setup

#### Registration

1. Launch the Label app
2. Tap "Create Account"
3. Enter your details:
   - Full Name
   - Email Address
   - Password (minimum 8 characters)
   - Phone Number (optional)
4. Review and accept:
   - [ ] I agree to the Terms & Conditions
   - [ ] I acknowledge the Privacy Policy
5. Tap "Sign Up"
6. Verify your email (check inbox)
7. Wait for administrator to assign permissions

**Note**: New users start with no permissions. Contact your administrator to request access to specific permit types.

#### Login

1. Open the Label app
2. Enter your email and password
3. Tap "Sign In"
4. (Optional) Enable biometric login for faster access

### Reset Password

**Forgot Password:**
1. Tap "Forgot Password?" on login screen
2. Enter your email address
3. Check your email for reset link
4. Follow link to set new password
5. Return to app and log in

### Permit Type Selector

At the top of the dashboard, you'll see a dropdown showing the current permit type:

- **Pool Permits**: Pool construction/renovation leads
- **Kitchen & Bath Permits**: Kitchen and bathroom remodeling leads
- **Roof Permits**: Roofing project leads

**To Switch Permit Types:**
1. Tap the permit dropdown at the top
2. Select desired permit type
3. Dashboard updates to show only those leads

**Note**: You'll only see permit types you have permission to view.

### Search & Filter

**Search:**
1. Tap the search icon (🔍)
2. Type to search:
   - Record ID
   - Name
   - Address
   - City
   - Phone
   - Email
3. Results update in real-time

**Filter by Status:**
- **All**: Show all leads
- **New**: Leads not yet contacted
- **Contacted**: Leads you've reached out to
- **Responded**: Leads who replied
- **Qualified**: Promising leads
- **Disqualified**: Not a good fit
- **Converted**: Successful conversions

**Sort Options:**
- Newest First (default)
- Oldest First
- Name A-Z
- Name Z-A
- City A-Z

### Pull to Refresh

Swipe down from the top of the lead list to refresh and fetch the latest leads.

## Templates

Templates help you send consistent, professional messages quickly.

### Template Categories

For each permit type, you can create templates for:

1. **Homeowner Email**: Professional emails to homeowners
2. **Homeowner Text**: SMS messages to homeowners
3. **Contractor Email**: B2B emails to contractors
4. **Contractor Text**: SMS alerts to contractors

### View Templates

1. Tap "Templates" tab at bottom
2. Select permit type from dropdown
3. Choose category (Email/Text, Homeowner/Contractor)

### Use Template

**From Templates Screen:**
1. Browse templates
2. Tap "Use" on desired template
3. Select lead to send to
4. Review and customize
5. Send

**From Lead Detail:**
1. Open lead
2. Tap "Email" or "Text"
3. Tap "Choose Template"
4. Select template
5. Template populates send form
6. Edit as needed
7. Send

### Create Template Manually

**If you have Manage Templates permission:**

1. Go to Templates tab
2. Select permit type and category
3. Tap "+ New Template" button
4. Fill in:
   - Template Name
   - Subject (for emails)
   - Body content
5. Use variables:
   - `{{firstName}}` - Contact first name
   - `{{lastName}}` - Contact last name
   - `{{fullName}}` - Full name
   - `{{address}}` - Property address
   - `{{city}}` - City
   - `{{state}}` - State
   - `{{permitType}}` - Permit type name
6. Tap "Save"

**Example Template:**

```
Subject: Your {{permitType}} Permit - Free Consultation

Hi {{firstName}},

I noticed you recently pulled a {{permitType}} permit for
your property at {{address}}.

We specialize in {{permitType}} projects in {{city}} and
would love to offer you a free consultation.

Reply to this email or call us at (555) 123-4567.

Best regards,
[Your Company]
```

### Generate Template with AI

**If you have Manage Templates permission:**

1. Go to Templates tab
2. Select permit type and category
3. Tap "Generate with AI" button
4. Choose AI provider:
   - OpenAI (ChatGPT)
   - Google Gemini
5. (Optional) Add custom instructions
6. Tap "Generate"
7. Review generated content
8. Edit as needed
9. Tap "Save as Template"

**AI Generation Tips:**
- Be specific in custom instructions
- Mention tone (professional, friendly, urgent)
- Specify key points to include
- Review carefully before saving
- Test with real lead before mass use

### Edit Template

1. Find template in list
2. Tap "Edit"
3. Make changes
4. Tap "Save"

**Note**: Changes affect future uses only. Already-sent messages are not changed.

### Copy Template

To create variation of existing template:

1. Find template
2. Tap "Copy"
3. Template duplicates with "(Copy)" suffix
4. Edit the copy as needed
5. Save

### Delete Template

1. Find template
2. Tap "Delete"
3. Confirm deletion
4. Template is permanently removed

**Warning**: Cannot be undone. If template was used previously, communication history still shows it was used.

## Communications

### Send Email

**From Lead Detail:**

1. Tap "Email" button
2. **Select Recipient**:
   - Choose from available email addresses
   - Lead may have up to 3 emails
3. **Choose Template** or write custom
4. **Review Email**:
   - Subject line
   - Body content
   - Variables are auto-filled
5. **Edit** if needed
6. Tap "Send Email"
7. Confirmation appears

**Email Tracking:**
- Sent status confirmed immediately
- Delivery status updates within minutes
- Opens tracked (if recipient enables images)
- Clicks tracked on links

### Send SMS

**From Lead Detail:**

1. Tap "Text" button
2. **Select Phone Number**:
   - Choose from available numbers (up to 3)
3. **Choose Template** or write custom
4. **Review Message**:
   - Character count shown
   - Messages over 160 chars = multiple segments
   - Cost increases with segments
5. **Edit** if needed
6. Tap "Send SMS"
7. Confirmation appears

**SMS Best Practices:**
- Keep under 160 characters when possible
- Include your name/company
- Provide opt-out option (e.g., "Reply STOP to opt out")
- Respect business hours
- Don't send too frequently

**SMS Tracking:**
- Delivery status updates in real-time
- Failed messages marked clearly
- Reply tracking (webhook required)

### Make Phone Call

1. Tap "Call" button
2. Select phone number
3. Device phone app opens
4. Call connects normally
5. Return to Label app after call
6. (Optional) Add notes about call outcome

**Note**: Calls are not tracked automatically. Add notes to document conversation.

### Communication History

View all past communications for a lead:

1. Open lead detail
2. Scroll to "Communication History"
3. Each entry shows:
   - Type (Email/SMS)
   - Timestamp
   - Status (Sent/Delivered/Opened/Clicked/Failed)
   - Template used (if any)
   - Sent by (user name)

**Status Indicators:**

**Email:**
- ⏳ Pending: Just sent
- ✅ Delivered: Reached inbox
- 📬 Opened: Recipient viewed
- 🔗 Clicked: Recipient clicked link
- ❌ Bounced: Failed delivery

**SMS:**
- ⏳ Pending: Sending
- ✅ Delivered: Received by phone
- ❌ Failed: Delivery failed

### Batch Communications

**If you have permission:**

Send to multiple leads at once:

1. From Dashboard, tap edit/select mode
2. Check boxes next to desired leads
3. Tap "Bulk Action" at bottom
4. Choose "Send Email" or "Send SMS"
5. Select template
6. Review recipients list
7. Confirm send

**Note**: Variables auto-fill for each recipient individually.

## Notifications

### Notification Center

Access notifications:

1. Tap "Notifications" tab (🔔)
2. View list of all notifications
3. Badge shows unread count

### Notification Types

**New Lead:**
- Triggered when new lead added to system
- Shows lead name and location
- Tap to open lead detail

**Policy Update:**
- Triggered when Terms or Privacy updated
- Requires re-acceptance
- Tap to review and accept

**Permission Change:**
- Triggered when your permissions change
- Shows what changed
- Tap for details

**System Alert:**
- Important system messages
- Admin announcements
- Maintenance notices

### Manage Notifications

**Mark as Read:**
1. Swipe notification left
2. Tap "Mark Read"

OR

1. Tap notification to open
2. Automatically marked read

**Mark All as Read:**
1. Tap "..." menu in Notifications
2. Select "Mark All Read"

**Delete Notification:**
1. Swipe notification left
2. Tap "Delete"

**Clear All:**
1. Tap "..." menu
2. Select "Clear All"
3. Confirm

### Push Notifications

**Enable:**
1. Go to Settings
2. Tap "Notifications"
3. Toggle "Push Notifications" ON
4. Grant permission when prompted

**Configure:**
- **New Leads**: Notify when leads added
- **Messages**: Notify about replies
- **Updates**: System and policy updates
- **Quiet Hours**: Set do-not-disturb times

**Disable:**
1. Settings > Notifications
2. Toggle OFF
3. You'll still see in-app notifications

### Badge Count

The app icon shows a red badge with number of:
- Unread notifications
- New/uncontacted leads (configurable)

**Reset Badge:**
- Badge clears when you view Dashboard
- Or mark notifications as read

## Troubleshooting

### Common Issues

**Cannot Login:**
- Check email/password spelling
- Ensure account activated (check email)
- Try password reset
- Check internet connection

**Not Seeing Leads:**
- Verify you have View permission
- Check permit type selected
- Try pull-to-refresh
- Contact administrator

**Cannot Send Email/SMS:**
- Verify you have permission
- Check internet connection
- Ensure valid email/phone number
- Check communication history for errors

**App Crashes:**
- Update to latest version
- Restart device
- Clear app cache
- Reinstall app

**Notifications Not Working:**
- Check notification permissions
- Ensure push notifications enabled
- Check device settings
- Re-login to app

### Get Help

1. Settings > About > Contact Support
2. Include:
   - Your email
   - Device type (iPhone/Android)
   - App version
   - Screenshot of issue
   - Description of problem

## Best Practices

### Lead Management

1. **Review Daily**: Check dashboard daily for new leads
2. **Respond Quickly**: Contact new leads within 24 hours
3. **Document Everything**: Use notes for important details
4. **Update Status**: Keep status current for team visibility
5. **Follow Up**: Set reminders for follow-up contacts

### Communication

1. **Personalize**: Edit templates to match each lead
2. **Timing**: Send during business hours (9am-6pm)
3. **Test First**: Test templates before bulk sending
4. **Track Results**: Review open/click rates
5. **Respect Opt-Outs**: Honor unsubscribe requests immediately

### Template Creation

1. **Be Clear**: Use simple, direct language
2. **Value Proposition**: Lead with benefits
3. **Call to Action**: Always include clear next step
4. **Test Variables**: Ensure all variables work
5. **Keep Updated**: Refresh templates regularly

### Data Security

1. **Strong Password**: Use unique, complex password
2. **Logout**: Log out on shared devices
3. **Biometric**: Enable for secure, quick access
4. **Don't Share**: Never share your login credentials
5. **Report Issues**: Alert admin of suspicious activity
